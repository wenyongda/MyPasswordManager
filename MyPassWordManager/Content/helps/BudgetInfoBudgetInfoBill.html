预算信息操作指南

1.预算信息界面

点击【市场管理】的菜单栏，选择左侧菜单栏【预算】下的【预算信息】，单击后进入预算信息界面，如下图所示。

找到项目登记年份下的具体投标信息下的包，可以查看该包的预算信息。其中投标负责人只能查看自己负责的投标信息下的包。

如果该包创建时选择了物资成本和实施方案，那么只有物资成本和实施方案全都有发布的版本后，才可以在预算信息界面看见该包。

如果该包创建时只选择了物资成本或者实施方案，那么只要物资成本或者实施方案有发布的版本后，就可以在预算信息界面看见该包。

2.预算信息的操作

①汇总

选中投标信息或者具体的包信息，可以查看该投标信息或者选中包的汇总信息。点击左下角的按钮，选中【汇总】标签页即可。

其中，

一. 人工总成本比例：（(40.自行实施成本 + 44.安装调试 + 20.02.电气外包 + 20.01.技术外包 + 20.04.安装调试外包+20.05人力外包+20.06培训外包) / 总成本

1. 自行人工实施成本比例：(40.自行实施成本 + 44.安装调试) / 总预算

2. 人力外包实施成本比例：(20.02.电气外包 + 20.01.技术外包 + 20.04.安装调试外包++20.05人力外包+20.06培训外包) / 总成本

(1)电气外包比例：20.02.电气外包 / 总预算

(2)技术外包比例：20.01.技术外包 / 总预算

(3)机械安装及调试外包比例：20.04.安装调试外包+20.06培训外包 / 总成本

二. 材料总成本比例：(21.机械外购成本 +22气动液压外购件+ 23.电气外购件 + 31.原材料 + 41.制造检测费用 + 42.外协加工费+20.03整体外包) /总成本

1. 机械外购成本比例：21.机械外购成本/总预算

2. 电气外购成本比例：23.电气外购件/总预算

3. 气动液压外购件成本比例：22.气动外购件/总预算

4. 制造成本（原材料+制造+外协）比例：(31.原材料 + 41.制造检测费用 + 42.外协加工费) / 总预算

5. 整体外包成本比例： 20.03/总预算

三. 包装运输比例：43.包装运输成本/总预算

四. 备件及其他比例：(45.备件 + 46.其他) /总预算

②切换版本

选中包后，默认显示的内容是当前选中版本的物资成本和实施方案内容。可以点击该包左下角的按钮，选择版本标签页，然后可以选择其他版本的物资成本和实施方案，那么预算信息界面就会显示选择的物资成本和实施方案对应的预算信息。

③全部展开&全部折叠

选中包后，点击右上角的【全部展开】或者【全部折叠】按钮，可以对预算信息全部展开到六位码下的具体明细，或者把展开的明细全部折叠上。

④切换币别

选中包后，点击右上角的币别，可以对预算信息进行币别的切换。选择其他币别后，该预算信息下所有的本位币按照当前汇率进行转换。该币别列表下的币别是在物资成本中进行维护的。

⑤界面编辑

在预算信息界面，可以对单价系数、数量系数和总价系数进行编辑。编辑单价系数和数量系数后，对应的单价和数量会进行更改。单价=原单价*单价系数，数量=原数量*数量系数。更改总价系数后，会根据配置中的总价系数对应规则进行更改，如果总价系数对应改则中，更改总价系数对应的是数量，那么更改总价系数后，在预算信息界面就会对数量进行变更，其它的都对单价进行变更。

其中市场部科长和部长有权限对单价系数和数量系数进行编辑。市场部的主管领导有权限对总价系数进行编辑。

根据实际业务需求，预算信息中的所有的系数都可以设置为0。然后在物资成本中重新新增版本进行录入，新增的物资版本发布后，会把上一版中录入的系数自动带入的预算信息中，新添加的物资的系数会默认为“1”。

⑥批量编辑系数

选中单个包后，可以批量编辑该包的预算信息的单价系数、数量系数和总价系数。点击左下角的按钮，在【系数】选项中，选中想要调整的系数，即可对该系数进行批量调整。

但是只有市场部的主管领导具有批量编辑系数的权限。

其中，全局设定可以选择单价系数、数量系数和总价系数。选择后，填写想要降的百分比，点击【统一设定】按钮，预算信息界面的系数就会被批量更改，然后点击【保存】按钮即可对设定的系数进行保存。

举例说明：预算信息中的总价系数是“1”，在全局设定中选择总价系数后，填写20，那么预算信息中的总价系数就会变为“0.8”。

其中，批量设定可以选择物资成本和实施方案，然后批量对物资成本或者实施方案的单价系数、数量系数和总价系数进行设定。选择物资成本或者实施方案后，再选择想要调整的系数，然后填写想要调整的比例，点击【统一设定】按钮，预算信息界面的物资成本或者实施方案中的系数就会被批量更改，然后点击【保存】按钮即可对设定的系数进行保存。

举例说明：预算信息中的总价系数是“0.8”，在全局设定中选择物资成本的总价系数后，填写0.8，那么预算信息中的所有物资成本的总价系数就会变为“0.64”，即原总价系数*0.8。

⑦附件

选中单个包后，点击左下角的按钮，选择【附件】标签页，附件中会显示实施方案中上传的附件。选中附件后，点击【下载】按钮，即可将附件下载到电脑中进行查看。

⑧测算

选中投标信息或者单个包后，点击左下角的按钮，选择【测算】标签页，可以对毛利率进行测算。其中预算总额显示的当前选中的包的预算信息中的总金额，汇率默认显示1.0000。可以进行更改，然后手动输入合同额后，系统可以自动计算毛利率。

其中，毛利率=1-（预算总额*汇率/合同额）

⑨导出

选中投标信息或者单个包后，点击预算信息上的【导出】按钮，即可对所显示的预算信息进行导出，其中导出的数据是进行了加密处理的，需要使用其他软件对导出的数据进行查看。

登陆可以打开导出内容的软件，输入用户名和密码后，点击【登陆】按钮。

登陆成功后，点击【文件查看】按钮，选择想要打开的文件后，系统会自动将导出的加密文件进行打开，允许对打开的文件进行修改，点击【保存】按钮即可。

⑩下发

选中多个或者单个包后，点击【下发】按钮，即可对选中的包进行下发。下发时，要求选中的包必须已经关联项目。该包下发后，包会自动关闭，并且的状态是已中标的状态。下发后，该包在【投标中】的状态下消失，变为【已关闭】。

其中，只有公司授权人有权限进行下发。
